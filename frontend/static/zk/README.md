# ZK Circuit Artifacts

Place the compiled circuit files here:

1. `citizen_credential.wasm`
2. `citizen_credential_final.zkey`

These are generated by compiling `blockchain/circuits/citizen_credential.circom`.

## How to generate:

```bash
cd blockchain
# Compile circuit
circom circuits/citizen_credential.circom --r1cs --wasm --sym
# (Setup trusted setup to get .zkey - see snarkjs docs)
# Copy files to this folder
cp citizen_credential_js/citizen_credential.wasm ../frontend/static/zk/
cp citizen_credential_final.zkey ../frontend/static/zk/
```
